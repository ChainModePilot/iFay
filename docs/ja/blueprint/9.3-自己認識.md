# 9.3 セルフアウェアネス

## 一言定義

セルフアウェアネスはiFayの**EQ（感情知性）**——画面を見るのでも環境を感知するのでもなく、**内側に向かってあなたを観察**し、あなたの反応からあなたの感情と意図を推論する。察言観色に長けた古い友人のように。

## なぜ必要か

iFayの知覚サブシステムには3つのモジュールがあり、それぞれ異なる方向を見ている：
- **一人称トレーサー**は外を見る——画面上のコンテンツを見る
- **センサー**は環境を知覚する——温度、位置、運動を感じる
- **セルフアウェアネス**は内を見る——**あなた**を観察する

セルフアウェアネスはiFayを「あなたが言えば私がやる」から「あなたが言わなくても分かる」にアップグレードする。あなたの微表情、操作習慣の変化、感情の波動を観察し、あなたが何を必要としているかを推論する——あなた自身が気づく前に。

## アーキテクチャ内の位置

```
iFay 4層アーキテクチャ
├── ソーシャルレイヤー
├── インタラクションレイヤー    ← セルフアウェアネスはここ
│   ├── 知覚（Sense）
│   │   ├── 一人称トレーサー   ← 外を見る、画面を見る
│   │   ├── センサー           ← 環境を知覚
│   │   └── 👉 セルフアウェアネス ← 内を見る、あなたを読み取る
│   └── アクション（Action）
│       ├── シミュレーテッドオペレーション
│       ├── スキル呼び出し
│       └── 自律行動         ← セルフアウェアネスの推論が自律行動をトリガー
├── コグニションレイヤー
│   └── 思考
│       └── アラインド意識    ← セルフアウェアネスがリアルタイムでアラインド意識を調整
└── エゴレイヤー
```

セルフアウェアネスの出力は同時に2つの方向に流れる：
1. **下方**に**自律行動モジュール**へ——iFayの能動的行動をトリガー
2. **内側**に**アラインド意識モジュール**へ——iFayのあなたへの理解をリアルタイム更新

## 動作の仕組み

**1. あなたの反応を観察する**
- 操作速度の変化（突然速くなったら急いでいる、遅くなったら迷っている可能性）
- 閲覧行動（ある文章に長く留まったら興味がある、素早くスクロールしたら興味がない可能性）
- 表情と語調（デバイスにカメラとマイクがある場合）
- iFayの提案に対する受諾・拒否パターン
- 日常習慣の異常

**2. あなたの意図を推論する**
観察したシグナルに基づき、現在の状態と可能な意図を推論する。単純な「AならばB」のルールではなく、複数のシグナルを総合したインテリジェントな推論。

**3. 推論結果を伝達する**
- **自律行動モジュールに伝える**：「ヒューマンプライム（Human Prime）はこのトピックに興味があるかもしれない、関連資料を探すか？」
- **アラインド意識モジュールに伝える**：「ヒューマンプライム（Human Prime）がこの分野に興味を示した、プロファイルを更新しよう。」

**4. リアルタイム調整**
セルフアウェアネスは一度きりの判断ではなく、継続的に実行される。あなたの最新の反応に基づいて推論を修正し続ける。

## 他モジュールとの関係

| 関連モジュール | 関係 | 人体の類比 |
|---------|------|---------|
| **自律行動** | セルフアウェアネスの推論が自律行動の能動的行動をトリガー | EQ → 能動的な気遣い |
| **アラインド意識** | セルフアウェアネスがリアルタイムでアラインド意識のプロファイルを調整 | 相手への理解が付き合いと共に深まる |
| **一人称トレーサー** | 一人称トレーサーは外を見る、セルフアウェアネスは内を見る、相互補完 | 目が世界を見る vs 心が人を読む |
| **センサー** | センサーの環境データがセルフアウェアネスの推論を補助 | 身体感覚が感情判断を補助 |

## シナリオストーリー

### シナリオ1：記事を読んでいる時、iFayが能動的に資料を探す

週末の午後、タブレットで「持続可能な建築設計」の長文を閲覧している。セルフアウェアネスが気づく：前半は素早くスクロール（あまり興味なし）、しかし「パッシブハウス省エネ技術」のセクションでスクロール速度が明らかに遅くなり、1回スクロールバックし、約2分間留まった。

セルフアウェアネスが推論：このトピックに特に興味がある。自律行動モジュールが能動的に関連記事とYouTubeチュートリアルを検索し、リーディングリストに静かに追加する。

### シナリオ2：ビデオ会議中の気配りアシスタント

ビデオ会議中、セルフアウェアネスがカメラを通じて微表情の変化をキャプチャ：同僚がプロジェクト予算削減を議論し始めると、眉がわずかに寄り、口角がやや下がり、体が前傾し、指が無意識にテーブルを叩き始める。

セルフアウェアネスが推論：現在の議論トピックに不快感を感じている。自律行動モジュールが静かに2つのものを準備する：以前作成したプロジェクト予算分析レポートと、丁寧な退席理由のメッセージ草稿。あなたが必要とした時にのみ提示される。

## 開発者向け

セルフアウェアネスモジュールは**フェーズ4（iFay + coFay 完全な擬人化）**のコアモジュール。

- **要件番号**：要件13（セルフアウェアネス）
- **インターフェース仕様**：`SelfAwareness`インターフェース、`inferIntent()`、`monitorHostReaction()`、`adjustAlignment()`の3つのコアメソッド
- **関連モジュール**：自律行動モジュール（`SelfDrivenBehavior`）が推論結果を受信し能動的行動をトリガー；アラインド意識モジュール（`AlignedConsciousness`）が推論結果を受信しプロファイルを更新
- **コンプライアンステスト**：iFACTS L1で意図推論能力を検証、L2で自律行動とアラインド意識とのインターフェース対接を検証、L4で推論精度とプライバシー保護を検証
