# 11.3 アラインド意識

## 一言定義

アラインド意識はiFayのあなたに対する**完全な理解**——Egoモデルがi Fayの「性格」なら、アラインド意識はiFayのあなたという人間に対する深い認知——あなたの価値観、嗜好、習慣、境界。iFayのすべての行動の基準線である。

## なぜ必要か

iFayのすべての行動には「基準線」が必要——この件はすべきか？どうすべきか？どの程度まですべきか？この基準線がアラインド意識が提供するヒューマンプライム（Human Prime）プロファイルである。

## アーキテクチャ内の位置

```
iFay 4層アーキテクチャ
├── インタラクションレイヤー
│   ├── 知覚（Sense）
│   │   └── セルフアウェアネス  ← リアルタイムで観察 → アラインド意識を更新
├── コグニションレイヤー    ← アラインド意識はここ
│   ├── 思考（Thought）
│   │   ├── パーソナルデータヒープ  ← ここからデータを発掘してあなたを理解
│   │   ├── 外部知識
│   │   └── 👉 アラインド意識      ← あなたへの完全な理解
│   └── スキル（Skill）
│       └── 内部スキル             ← アラインド意識で行動を制約
└── エゴレイヤー
    └── Egoモデル                  ← アラインド意識はEgoの入力
```

## 動作の仕組み

プロファイルの次元：**価値志向、興味・嗜好、習慣パターン、認知境界、スキル境界、権限境界、ワークスタイル**。

**3つの方法**で確立・更新される：

**1. データマイニング——あなたのデータからあなたを理解する**
パーソナルデータヒープから行動パターンを発掘する。被動的・持続的な方式。

**2. リアルタイム調整——あなたの反応から読み取る**
セルフアウェアネスモジュールがリアルタイムであなたの反応を観察し、アラインド意識に伝達する。主動的・即時的な方式。

**3. 手動定義——あなたが直接iFayに伝える**
「午前10時前に会議を入れないで」など。明確・即時的な方式。

3つの方式が相互補完：データマイニングが長期トレンドを提供し、リアルタイム調整が即時変化をキャプチャし、手動定義が明確な境界を設定する。

## 他モジュールとの関係

| 関連モジュール | 関係 | 人体の類比 |
|---------|------|---------|
| **パーソナルデータヒープ** | データを発掘してプロファイルを構築 | 記憶から「自分はどんな人か」をまとめる |
| **セルフアウェアネス** | リアルタイムで反応を観察しプロファイルを更新 | 察言観色 → 理解を更新 |
| **Egoモデル** | プロファイルをEgoモデルに提供し個性制約の入力とする | あなたへの理解 → iFayの性格を形成 |
| **内部スキル** | 内部スキルに行動制約基準を提供 | あなたへの理解 → 何ができて何ができないかを決定 |
| **自律行動** | 自律行動が「この件を能動的にすべきか」を判断するのを支援 | あなたの境界を理解 → 行動範囲を決定 |

## シナリオストーリー

### シナリオ1：注文データからベジタリアンへの転向を発見

過去3か月の注文記録を分析し、肉類メニューの比率が70%→30%→0%に変化したことを発見。プロファイルを更新し、以降のすべての関連行動を自動調整する。

### シナリオ2：直接ルールを伝える

「午前10時前に会議を入れないで」——この手動定義が即座にプロファイルに反映され、すべての関連行動に影響する。

## 開発者向け

アラインド意識は**フェーズ4（iFay + coFay 完全な擬人化）**のコアモジュール。

- **要件番号**：要件16（アラインド意識）
- **インターフェース仕様**：`AlignedConsciousness`インターフェース、`getProfile()`、`mineFromDataHeap()`、`adjustFromAwareness()`、`manualUpdate()`の4つのコアメソッド
- **プロファイル次元**：`values`、`preferences`、`habits`、`cognitiveBoundaries`、`skillBoundaries`、`permissionBoundaries`、`workingStyle`
- **3つの更新方式**：パーソナルデータヒープからの発掘（被動・持続）、セルフアウェアネスによるリアルタイム調整（主動・即時）、ヒューマンプライム（Human Prime）の手動定義（明確・即時）
- **iFay Profileとの関係**：ヒューマンプライム（Human Prime）プロファイル（HostProfile）はiFay Profileのサブセットかつ入力ソース
- **コンプライアンステスト**：iFACTS L1で3つのプロファイル更新方式を検証、L2でEgoモデルと内部スキルへのプロファイル伝達インターフェースを検証、L4でプロファイル更新が行動制約に与える実際の影響を検証
