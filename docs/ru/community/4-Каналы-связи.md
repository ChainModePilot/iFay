# 4. Каналы связи

Сообщество iFay предоставляет несколько каналов связи для удовлетворения потребностей в сотрудничестве в различных сценариях.

## GitHub Discussions

GitHub Discussions — это основная платформа асинхронного обсуждения сообщества iFay, подходящая для:

- Публичного обсуждения SEP (Specification Enhancement Proposals)
- Углублённого изучения технических вопросов
- Запросов на функциональность и обсуждений дизайна
- Повседневного общения внутри рабочих групп

Мы рекомендуем проводить важные технические обсуждения на GitHub Discussions, чтобы обеспечить отслеживаемость и возможность поиска содержания обсуждений.

## GitHub Issues

GitHub Issues используется для отслеживания конкретных задач и управления проблемами:

- Отчёты об ошибках
- Конкретные задачи разработки
- Предложения по улучшению документации
- Проблемы с тестовыми случаями iFACTS

При создании Issue, пожалуйста, предоставьте подробные шаги воспроизведения или контекстную информацию, чтобы помочь мейнтейнерам быстро локализовать и устранить проблему.

## Discord

Discord — это платформа для общения в реальном времени сообщества iFay. Каждая рабочая группа имеет свой выделенный канал:

- `#general`: Общее обсуждение
- `#wg-fayid`: Рабочая группа FayID
- `#wg-ego`: Рабочая группа Ego
- `#wg-protocols`: Рабочая группа протоколов
- `#wg-fayger`: Рабочая группа FayGer
- `#wg-ifacts`: Рабочая группа iFACTS
- `#wg-docs`: Рабочая группа документации
- `#wg-community`: Рабочая группа сообщества

Discord подходит для быстрого обмена информацией и неформальных обсуждений. Однако обратите внимание, что важные технические решения должны фиксироваться на GitHub для обеспечения прозрачности и отслеживаемости.

## Регулярные встречи

- **Ежемесячная встреча Core Maintainers**: Проводится раз в месяц, проходит публично и открыта для наблюдения всеми членами сообщества. Повестки дня и протоколы встреч публикуются на GitHub.
- **Встречи рабочих групп**: Каждая рабочая группа может устанавливать собственную частоту и формат встреч, рекомендуется проводить не менее одной синхронизационной встречи в месяц.

## Список рассылки

Список рассылки используется для важных объявлений и уведомлений:

- Уведомления о выпуске новых версий
- Уведомления об изменении статуса SEP
- Объявления о важных решениях по управлению
- Уведомления о мероприятиях сообщества

Подписка на список рассылки гарантирует, что вы не пропустите важную информацию.

## Кодекс поведения

Все каналы связи следуют Кодексу поведения iFay. Мы стремимся создать дружелюбную, инклюзивную и профессиональную среду сообщества. Пожалуйста, сохраняйте уважение и конструктивность во всех взаимодействиях.

## Язык

Английский является основным рабочим языком сообщества iFay, но мы приветствуем участие на других языках. Если вам удобнее использовать русский, китайский, немецкий или другой язык, пожалуйста, не стесняйтесь — члены сообщества помогут с переводом, чтобы ваши идеи были точно переданы.

Наша цель — обеспечить, чтобы язык не был барьером для участия в сообществе iFay.
