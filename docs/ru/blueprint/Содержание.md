- #### Руководство разработчика
  - Разработчики open-source проекта
    - Модель данных
    - FayGer (виртуальная среда выполнения)
      - Кросс-языковые инструменты сборки
      - Кросс-платформенный контейнер выполнения
    - Протоколы
      - [Протокол Faying](https://github.com/ChainModePilot/Faying-Protocol/wiki)
      - [Протокол интерактивного диалога](https://github.com/ChainModePilot/Interactive-Conversation-Protocol/wiki)
      - [Протокол Телепатии](https://github.com/ChainModePilot/Telepathy-Protocol/wiki)
      - [Протокол управления полномочиями (CAP)](https://github.com/ChainModePilot/Control-Authority-Protocol/wiki)
      - [Протокол туннелирования данных (DTP)](https://github.com/ChainModePilot/Data-Tunnel-Protocol/wiki)
      - [Протокол обмена навыками (SSP)](https://github.com/ChainModePilot/Skill-Sharing-Protocol/wiki)
  - Разработчики приложений
    - Поддержка Faying с iFay в приложениях
    - Поддержка Faying с iFay в аппаратных устройствах
    - Облачное развёртывание среды выполнения FayGer
    - Обмен навыками с iFay
    - Модульная мультимодальная передача сообщений
  - Разработчики сервис-провайдеров
    - Разработка и создание сущностей iFay
