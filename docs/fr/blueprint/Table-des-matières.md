- #### Guide du développeur
  - Développeurs du projet open source
    - Modèle de données
    - FayGer (Environnement d'exécution virtuel)
      - Outils de compilation multi-langages
      - Conteneur d'exécution multi-plateformes
    - Protocoles
      - [Protocole Faying](https://github.com/ChainModePilot/Faying-Protocol/wiki)
      - [Protocole de conversation interactive](https://github.com/ChainModePilot/Interactive-Conversation-Protocol/wiki)
      - [Protocole de Télépathie](https://github.com/ChainModePilot/Telepathy-Protocol/wiki)
      - [Protocole d'autorité de contrôle](https://github.com/ChainModePilot/Control-Authority-Protocol/wiki)
      - [Protocole de tunnel de données](https://github.com/ChainModePilot/Data-Tunnel-Protocol/wiki)
      - [Protocole de partage de compétences](https://github.com/ChainModePilot/Skill-Sharing-Protocol/wiki)
  - Développeurs d'applications
    - Support APP pour le Faying avec iFay
    - Support des périphériques matériels pour le Faying avec iFay
    - Déploiement cloud de l'environnement d'exécution FayGer
    - Partage de compétences avec iFay
    - Messagerie modulaire multimodale
  - Développeurs de fournisseurs de services
    - Développement et construction d'entités iFay
