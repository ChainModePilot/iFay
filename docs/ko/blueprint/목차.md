- #### 개발자 가이드
  - 오픈소스 프로젝트 개발자
    - 데이터 모델
    - FayGer(가상 런타임 환경)
      - 크로스 언어 빌드 도구
      - 크로스 플랫폼 실행 컨테이너
    - 프로토콜
      - [Faying 프로토콜](https://github.com/ChainModePilot/Faying-Protocol/wiki)
      - [인터랙티브 대화 프로토콜](https://github.com/ChainModePilot/Interactive-Conversation-Protocol/wiki)
      - [텔레파시 프로토콜(Telepathy Protocol)](https://github.com/ChainModePilot/Telepathy-Protocol/wiki)
      - [CAP(Control Authority Protocol)](https://github.com/ChainModePilot/Control-Authority-Protocol/wiki)
      - [DTP(Data Tunnel Protocol)](https://github.com/ChainModePilot/Data-Tunnel-Protocol/wiki)
      - [SSP(Skill Sharing Protocol)](https://github.com/ChainModePilot/Skill-Sharing-Protocol/wiki)
  - 애플리케이션 개발자
    - APP이 iFay와 Faying 지원
    - 하드웨어 디바이스가 iFay와 Faying 지원
    - 클라우드 배포 FayGer 런타임 환경
    - iFay와 스킬 공유
    - 모듈화 멀티모달 메시지
  - 서비스 제공자 개발자
    - iFay 엔티티 개발 및 구축
