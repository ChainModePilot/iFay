# 9. 인터랙션 레이어 — 감지

인터랙션 레이어의 감지 서브시스템은 iFay의 "감각 기관"—iFay가 화면을 보고, 환경을 감지하고, 당신의 의도를 읽을 수 있게 하여, 후속 동작과 인지에 정보 입력을 제공합니다.

---

## 9.1 일인칭 추적기

### 한 줄 정의

일인칭 추적기는 iFay의 **눈과 귀**—당신이 보는 화면, 듣는 소리를 iFay도 동일하게 볼 수 있게 합니다.

### 어떻게 작동하는가

1. **화면 캡처** — 화면의 모든 시각적 콘텐츠를 캡처
2. **오디오 캡처** — 인터페이스의 알림음이나 음성을 캡처
3. **실시간 변화 추적** — 조작 후 화면 변화를 즉시 추적
4. **손-눈 협응** — 시뮬레이션 조작 모듈과 긴밀하게 연계
5. **"보이지 않음" 보고** — 화면 취득 불가 시 코그니션 레이어에 보고

### 다른 모듈과의 관계

| 관련 모듈 | 관계 | 인체 비유 |
|---------|------|---------|
| **시뮬레이션 조작** | 긴밀 결합, 손-눈 협응 | 눈 ↔ 손 |
| **센서** | 같은 감지 서브시스템, 다른 분담 | 눈 vs 전신 신경 |
| **자기인식** | 방향이 다름: 밖을 봄 vs 안을 봄 | 눈 vs EQ |

### 개발자 참고

- **인터페이스 사양**: `FirstPersonTracer`, `captureVisual()`, `captureAudio()`, `trackChanges()`, `getPerceptionStatus()` 포함
- **적합성 테스트**: iFACTS L1에서 시각 캡처 능력 검증, L2에서 시뮬레이션 조작과의 손-눈 협응 인터페이스 검증

---

## 9.2 센서

### 한 줄 정의

센서는 iFay의 **신경계**—온도, 위치, 심박수, 빛 등 주변 환경의 모든 변화를 감지할 수 있게 합니다.

### 어떻게 작동하는가

**1. 브리징** — CAP/DTP 프로토콜을 통해 디바이스 센서 데이터를 통합 포맷으로 변환
**2. 조절** — 장면과 수요에 따라 각 센서의 감도를 동적으로 조절
**3. 확장** — 새로운 센서 유형은 디바이스 드라이버 허브를 통해 통합 가능

### 다른 모듈과의 관계

| 관련 모듈 | 관계 | 인체 비유 |
|---------|------|---------|
| **디바이스 드라이버 허브** | 하드웨어 인터페이스 관리 | 신경 말단 → 신경 전도로 |
| **개인 데이터 힙** | 센서 데이터 저장 | 감각 신호 → 기억 저장 |
| **CAP / DTP 프로토콜** | 데이터 브리징 실현 | 신경 신호의 전송 프로토콜 |

### 개발자 참고

- **인터페이스 사양**: `SensorModule`, `registerSource()`, `adjustSensitivity()`, `getDataStream()`, `getActiveStatus()` 포함
- **적합성 테스트**: iFACTS L1에서 감도 조절 능력 검증, L2에서 디바이스 드라이버 허브와 개인 데이터 힙과의 인터페이스 검증

---

## 9.3 자기인식

### 한 줄 정의

자기인식은 iFay의 **EQ(감성 지능)**—화면을 보지 않고, 환경을 감지하지 않고, **당신의 내면을 관찰**하여 당신의 반응에서 감정과 의도를 추론합니다.

### 어떻게 작동하는가

1. **반응 관찰** — 조작 속도 변화, 브라우징 행동, 표정, 제안에 대한 수용/거부 패턴을 모니터링
2. **의도 추론** — 여러 신호를 종합적으로 지능적으로 추론
3. **추론 결과 전달** — 자율 행동 모듈과 정렬 의식 모듈에 동시 전달
4. **실시간 조정** — 최신 반응에 기반하여 추론을 지속적으로 수정

### 다른 모듈과의 관계

| 관련 모듈 | 관계 | 인체 비유 |
|---------|------|---------|
| **자율 행동** | 추론이 자율 행동을 트리거 | EQ → 자발적 배려 |
| **정렬 의식** | 실시간으로 휴먼 프라임 프로필 업데이트 | 이해가 깊어짐 |
| **일인칭 추적기** | 밖을 봄 vs 안을 봄, 상호 보완 | 눈 vs 마음 |

### 개발자 참고

- **인터페이스 사양**: `SelfAwareness`, `inferIntent()`, `monitorHostReaction()`, `adjustAlignment()` 포함
- **적합성 테스트**: iFACTS L1에서 의도 추론 능력 검증, L4에서 추론 정확도와 프라이버시 보호 검증
