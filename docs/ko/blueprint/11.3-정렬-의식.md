# 11.3 정렬 의식

## 한 줄 정의

정렬 의식은 iFay가 당신에 대해 가진 **완전한 이해**입니다—Ego 모델이 iFay의 "성격"이라면, 정렬 의식은 iFay가 당신이라는 사람에 대해 가진 깊은 인지입니다—가치관, 선호, 습관, 경계. iFay 모든 행동의 기준선입니다.

## 왜 필요한가

iFay의 모든 행동에는 "기준선"이 필요합니다: 이 일을 해야 하는가? 어떻게 해야 하는가? 어느 정도까지 해야 하는가? 이 기준선이 정렬 의식이 제공하는 휴먼 프라임(Human Prime) 프로필입니다.

## 작동 방식

정렬 의식이 유지하는 휴먼 프라임(Human Prime) 프로필은 여러 차원을 포함합니다: **가치 지향, 관심 선호, 습관 패턴, 인지 경계, 스킬 경계, 권한 경계, 작업 스타일**.

이 프로필은 **세 가지 방식**으로 구축되고 업데이트됩니다:

**1. 데이터 마이닝—당신의 데이터에서 당신을 이해**

개인 데이터 힙에서 행동 패턴을 마이닝합니다. 이 방식은 **수동적이고 지속적**입니다.

**2. 실시간 조정—당신의 반응에서 당신을 읽음**

자기인식 모듈이 실시간으로 반응을 관찰하고, 관찰 결과를 정렬 의식에 전달합니다. 이 방식은 **능동적이고 즉시적**입니다.

**3. 수동 정의—당신이 직접 iFay에게 말함**

일부 선호와 경계는 직접 말할 수 있습니다. 이 방식은 **명확하고 즉시적**입니다.

세 가지 방식이 상호 보완합니다: 데이터 마이닝이 장기 추세를 제공하고, 실시간 조정이 즉시 변화를 포착하며, 수동 정의가 명확한 경계를 설정합니다.

## 개발자를 위해

정렬 의식은 **4단계(iFay + coFay 전면 의인화)**의 핵심 모듈입니다.

- **요구사항 번호**: 요구사항 16(정렬 의식)
- **인터페이스 규격**: `AlignedConsciousness` 인터페이스, `getProfile()`, `mineFromDataHeap()`, `adjustFromAwareness()`, `manualUpdate()` 4개 핵심 메서드 포함
- **프로필 차원**: `values`, `preferences`, `habits`, `cognitiveBoundaries`, `skillBoundaries`, `permissionBoundaries`, `workingStyle`
- **적합성 테스트**: iFACTS L1이 세 가지 프로필 업데이트 방식을 검증, L4가 프로필 업데이트가 행동 제약에 미치는 실제 영향을 검증
